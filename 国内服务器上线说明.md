# 国内服务器上线说明

这份说明适合「中国大陆能打开」的用法：买一台大陆轻量服务器，然后用公网 IP 访问。

最终访问地址类似：

```text
http://你的服务器公网IP:8000
```

## 推荐购买

推荐买腾讯云或阿里云轻量应用服务器：

- 地域：上海、广州、北京、南京都可以，选离你们近的。
- 系统：Ubuntu 22.04 或 Ubuntu 24.04。
- 配置：2 核 2G 起步即可。
- 带宽：3M 起步，Excel 文件大就选 5M。

如果暂时不用域名，只用 `公网IP:8000`，一般不需要备案。

## 服务器安全组

买完服务器后，在云平台控制台打开防火墙/安全组，放行：

```text
TCP 22
TCP 8000
```

`22` 用来远程登录服务器，`8000` 用来访问网站。

## 连接服务器

在 Windows PowerShell 输入：

```powershell
ssh root@你的服务器公网IP
```

第一次会问是否继续，输入：

```text
yes
```

然后输入服务器密码。

## 安装 Docker

登录服务器后，复制执行：

```bash
apt update
apt install -y ca-certificates curl git
install -m 0755 -d /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg -o /etc/apt/keyrings/docker.asc
chmod a+r /etc/apt/keyrings/docker.asc
echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.asc] https://download.docker.com/linux/ubuntu $(. /etc/os-release && echo "${UBUNTU_CODENAME:-$VERSION_CODENAME}") stable" > /etc/apt/sources.list.d/docker.list
apt update
apt install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

检查 Docker：

```bash
docker --version
docker compose version
```

## 下载项目代码

执行：

```bash
cd /opt
git clone https://github.com/Friesasa/-.git xiaohongshu-ratecard-automation
cd xiaohongshu-ratecard-automation
```

如果 GitHub 在服务器上下载很慢，可以多试一次。

## 配置环境变量

复制模板：

```bash
cp .env.server.example .env
```

生成一个随机密钥：

```bash
openssl rand -hex 32
```

编辑 `.env`：

```bash
nano .env
```

至少修改：

```text
SECRET_KEY=刚才生成的随机密钥
REGISTER_INVITE_CODE=你自己设置的邀请码
```

如果有 OpenAI 或兼容模型 Key，再填：

```text
OPENAI_API_KEY=你的Key
```

保存 nano：

```text
Ctrl + O
回车
Ctrl + X
```

## 启动网站

执行：

```bash
docker compose up -d --build
```

查看状态：

```bash
docker compose ps
```

看到 `xiaohongshu-ratecard-web` 是 running 就可以。

## 打开网站

浏览器打开：

```text
http://你的服务器公网IP:8000
```

然后注册账号。注册时如果设置了邀请码，就填写 `.env` 里的 `REGISTER_INVITE_CODE`。

## 日常维护

查看日志：

```bash
docker compose logs -f
```

重启：

```bash
docker compose restart
```

更新代码：

```bash
cd /opt/xiaohongshu-ratecard-automation
git pull
docker compose up -d --build
```

## 数据保存在哪里

网站数据、上传 Excel、导出 Excel 都在 Docker 数据卷 `xhs_data` 里。只要不删除这个数据卷，重启服务器或重启容器都不会丢。

不要执行下面这种命令：

```bash
docker volume rm xiaohongshu-ratecard-automation_xhs_data
```

这会删除网站数据。
